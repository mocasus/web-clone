# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Active support  |
| < 1.0   | ❌ Not supported   |

## Reporting a Vulnerability

**Please do NOT open a public issue for security vulnerabilities.**

Instead, report privately via one of these channels:

- **Telegram (preferred):** [@rubuskap](https://t.me/rubuskap)
- **GitHub Security Advisory:** [Report privately](https://github.com/mocasus/web-clone/security/advisories/new)

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Suggested fix (if known)

### What to Expect

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 1 week
- **Fix timeline:** depends on severity (critical → days, low → next minor release)
- **Credit:** you'll be acknowledged in the release notes unless you prefer anonymity

## Scope

In scope:
- Code in this repository
- Default configurations
- Dependency vulnerabilities affecting this project

Out of scope:
- Third-party services referenced (Claude, OpenAI, etc.) — report to those vendors directly
- Issues in projects built _using_ this template — those belong in the downstream repo

Thanks for helping keep this project secure! 🔒
